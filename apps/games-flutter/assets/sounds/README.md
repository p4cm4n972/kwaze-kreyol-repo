# Sons pour Skrabb

Ce dossier contient les effets sonores pour le jeu Skrabb.

## Fichiers nécessaires:

### Effets sonores (format MP3):

1. **tile_place.mp3** - Son de placement d'une tuile sur le plateau
   - Durée recommandée: 0.1-0.3s
   - Son: clic doux, "toc" de bois

2. **validate.mp3** - Son de validation d'un coup
   - Durée recommandée: 0.3-0.5s
   - Son: "ding" positif, succès

3. **error.mp3** - Son d'erreur
   - Durée recommandée: 0.2-0.4s
   - Son: "buzz" court, négatif

4. **victory.mp3** - Son de victoire (fin de partie)
   - Durée recommandée: 1-3s
   - Son: fanfare joyeuse, célébration

5. **shuffle.mp3** - Son de mélange du chevalet
   - Durée recommandée: 0.3-0.5s
   - Son: brassage de tuiles

6. **undo.mp3** - Son d'annulation
   - Durée recommandée: 0.2-0.3s
   - Son: "whoosh" inversé

### Musique de fond (optionnel):

7. **background_music.mp3** - Musique d'ambiance
   - Durée: boucle de 30-120s
   - Style: musique créole relaxante, acoustique

## Notes:

- Le service audio gère gracieusement l'absence de fichiers
- Le jeu fonctionne sans sons si les fichiers n'existent pas
- Les sons peuvent être activés/désactivés dans les paramètres
- Volume par défaut: 30-60% selon le type de son

## Sons Domino (dossier domino/)

1. **place_tile.mp3** - Placement d'un domino
2. **pass_turn.mp3** - Passer son tour
3. **round_win.mp3** - Victoire de manche
4. **game_win.mp3** - Victoire de partie
5. **blocked.mp3** - Jeu bloqué
